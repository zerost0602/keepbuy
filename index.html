동기화 및 알림 설정

실시간 클라우드 동기화

구글 Firebase Realtime Database와 연동하여 가족 스마트폰 간에 데이터를 실시간으로 동기화합니다.

Telegram 실시간 알림

살 물건 목록 수동 전송 및 매일 아침 9시 냉장고 유통기한 알림을 위해 텔레그램 설정을 입력하세요.

동기화 & 알림 설정 방법 안내
  1. 나만의 무료 Firebase DB 개설 순서:
    1. Firebase 콘솔에 구글 로그인 후 프로젝트 추가를 누릅니다.
    2. 프로젝트 이름을 설정하고 (애널리틱스 비활성화 추천), 프로젝트 만들기 완료를 대기합니다.
    3. 왼쪽 빌드(Build) 메뉴에서 Realtime Database를 선택하고 데이터베이스 만들기를 누릅니다.
    4. 위치는 임의 지정(예: 미국)하고 다음을 누른 뒤, 테스트 모드를 선택하고 활성화를 누릅니다.
    5. 생성된 데이터베이스 상단의 규칙(Rules) 탭으로 이동하여 아래 코드로 교체한 뒤 게시(Publish)를 누릅니다:
    {
  "rules": {
    ".read": true,
    ".write": true
  }
}
    6. 다시 데이터(Data) 탭으로 돌아와 상단의 링크 주소(예: https://...firebaseio.com/)를 복사하여 Firebase Database URL 칸에 넣고 저장하면 완료됩니다! (이 경우 방 번호 없이 가족끼리 100% 동일 화면을 완전히 공유합니다.)
  2. Telegram 알림 설정:
    - 텔레그램에서 @BotFather를 통해 봇 생성 후 고유 토큰 획득.
    - 가족 단톡방을 만들고 생성한 봇을 초대한 뒤, @myidbot을 단톡방에 임시 초대하여 단톡방 Chat ID(예: -100...) 확인.
